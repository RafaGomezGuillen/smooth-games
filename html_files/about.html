Smooth Match 3 is a game based on match 3 games puzzles. I create this webpage just for fun, not monetary purposes, obviously any kind of donation is great for me, but you are not forced to.

Smooth Match 3 design is inspired on Wordle, I like the design of that page and adapted it to this page. I wanted to try a minimalist desing as well as the Worlde game.

The whole code is not made for me, I would like to thank Rembound for his match 3 game code on Github. I added some functions such as moves left, countdown, high score, etc... But I want to emphasize that the whole merit is for Rembound.

Data and privacy

Game data is saved locally using localgroup Javascript function on the device where you play. I do not save any kind of personal information.

You are free to do whatever you want with this page.
