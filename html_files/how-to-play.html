The goal of this game is simple, reach the max score as fast as you can. The table has 7 different colors. If you connect 3 of them you will sum 100 points, but, if you connect 4 of them, for example, it will be 200 points and so on.

The maximum difficulty of this game is the moves left and time left that you have to play. You have a certain number of possible moves and time. So be careful with that.

Buttons

New Game

If you want to start a new game, press New Game button.

Show Moves

If you want to see the possible moves, press Show Moves button.

Enable IA Bot

If you want to play automatically, press Enable IA Bot button.

Titles

Moves left

 Moves left tells you how many moves you have.

Score

Score tells you the actual points you have.

Max score

Max score tells you the higher points that you have ever scored.

Time left

Time left tells you how many time you have to play the actual game.

Set moves and seconds

Set moves

 If you want to set a particular number of moves, introduce the value and press . It will be set in the new new game. Minimum: 10 and maximum: 500.

Set seconds

If you want to set a particular number of seconds, introduce the value and press . It will be set in the new new game. Minimum: 10 and maximum: 500.
